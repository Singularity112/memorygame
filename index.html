Basic usage

const matchGrid = new MatchGrid(document.getElementById("game"));

Custom size with custom cols, rows, amount and timer

const matchGrid2 = new MatchGrid(document.getElementById("game2"), { width: 1000, height: 600, columns: 2, rows: 2, time: 60 });

Custom card images

const matchGrid3 = new MatchGrid(document.getElementById("game3"), { columns: 3, rows: 4, symbols: [1, 2, 3, 4, 5, 6, 7, 8], });

Custom theme

const matchGrid3 = new MatchGrid(document.getElementById("game4"), { theme: { color: "red", cardBackColor: "green", cardColor: "blue", gap: "40px", }, });
